Gemini CLI Commands

Here is a list of commands, descriptions, and examples from the Gemini CLI GitHub page.

Basic Usage

gemini

  • Description: Starts the Gemini CLI in the current directory.
  • Example: 
    gemini

gemini --include-directories <directories>

  • Description: Includes multiple directories for Gemini to access.
  • Example: 
    gemini --include-directories ../lib,../docs

gemini -m <model_name>

  • Description: Uses a specific Gemini model.
  • Example: 
    gemini -m gemini-2.5-flash

Non-interactive Mode for Scripts

gemini -p "<prompt>"

  • Description: Gets a simple text response from Gemini.
  • Example: 
    gemini -p "Explain the architecture of this codebase"

gemini -p "<prompt>" --output-format json

  • Description: Gets a structured JSON output for advanced scripting.
  • Example: 
    gemini -p "Explain the architecture of this codebase" --output-format json

gemini -p "<prompt>" --output-format stream-json

  • Description: Gets newline-delimited JSON events for real-time event streaming.
  • Example: 
    gemini -p "Run tests and deploy" --output-format stream-json

Quick Examples (Interactive Prompts)

gemini > Write me a Discord bot...

  • Description: An interactive prompt to generate a Discord bot.
  • Example: 
    cd new-project/
gemini > Write me a Discord bot that answers questions using a FAQ.md file I will provide

gemini > Give me a summary...

  • Description: An interactive prompt to summarize recent code changes.
  • Example: 
    git clone https://github.com/google-gemini/gemini-cli
cd gemini-cli
gemini > Give me a summary of all of the changes that went in yesterday

Authentication

gemini (with environment variables)

  • Description: Starts Gemini CLI after configuring authentication via Google Cloud Project, Gemini API Key, or Vertex AI.
  • Examples:
    • Option 1: Login with Google 
      export GOOGLE_CLOUD_PROJECT="YOUR_PROJECT_ID"
gemini
    • Option 2: Gemini API Key 
      export GEMINI_API_KEY="YOUR_API_KEY"
gemini
    • Option 3: Vertex AI 
      export GOOGLE_API_KEY="YOUR_API_KEY"
export GOOGLE_GENAI_USE_VERTEXAI=true
gemini

Using MCP Servers (Custom Integrations)

@github

  • Description: Lists open pull requests using a configured GitHub MCP server.
  • Example: 
    @github List my open pull requests

@slack

  • Description: Sends a summary of commits to a Slack channel using a configured Slack MCP server.
  • Example: 
    @slack Send a summary of today's commits to #dev channel

@database

  • Description: Runs a database query using a configured database MCP server.
  • Example: 
    @database Run a query to find inactive users

Other Commands

/bug

  • Description: A command to report issues directly from the CLI.
