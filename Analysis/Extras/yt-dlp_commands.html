Important yt-dlp Commands

Command/Flag Description Example
(no flags) Downloads a video in the default best quality. This usually requires ffmpeg to merge the best video and audio streams. yt-dlp "VIDEO_URL"
-f "bestvideo+bestaudio/best" Explicitly selects the highest quality video and audio, then merges them. This is the default behavior. yt-dlp -f "bestvideo+bestaudio/best" "VIDEO_URL"
-F, --list-formats Lists all available formats (video and audio) for a given URL. yt-dlp -F "VIDEO_URL"
-f, --format Downloads a specific format. Use the format code from the list generated by -F. You can combine video and audio with '+'. yt-dlp -f 22 "VIDEO_URL"
yt-dlp -f 137+140 "VIDEO_URL"
-x, --extract-audio Extracts the audio from a video file. Requires ffmpeg or ffprobe. yt-dlp -x "VIDEO_URL"
--audio-format Specifies the audio format for extraction (e.g., mp3, wav, m4a). Used with -x. yt-dlp -x --audio-format mp3 "VIDEO_URL"
(playlist) Downloads an entire playlist. Just provide the playlist URL. yt-dlp "PLAYLIST_URL"
--playlist-items Downloads specific items from a playlist (e.g., items 1 to 3, item 7, and item 10). yt-dlp --playlist-items 1-3,7,10 "PLAYLIST_URL"
-o, --output Specifies a custom output filename or template. yt-dlp -o '%(title)s [%(id)s].%(ext)s' "VIDEO_URL"
--write-sub Downloads available subtitles for the video. yt-dlp --write-sub "VIDEO_URL"
--embed-subs Embeds subtitles into the video file (for formats like mp4, mkv). yt-dlp --embed-subs "VIDEO_URL"
-a, --batch-file Downloads all URLs listed in a text file. yt-dlp -a urls_to_download.txt
--limit-rate Limits the download speed (e.g., 500K, 1.5M). yt-dlp --limit-rate 1M "VIDEO_URL"
--simulate, -s Performs a dry run. It doesn't download anything but shows what would happen. yt-dlp -s "VIDEO_URL"
-j, --dump-json Dumps the video's metadata as a single JSON line. Does not download the video. yt-dlp -j "VIDEO_URL"
